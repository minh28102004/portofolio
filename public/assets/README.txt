Public asset structure:

- `assets/animations/`
  Use for hero GIFs and motion visuals used directly in the UI.
- `assets/avatars/`
  Default or reusable avatar images.
- `assets/brand/`
  Small logos and brand marks such as Spotify.
- `assets/documents/`
  Downloadable files that do not need to stay at the public root.
- `assets/lottie/`
  Raw Lottie JSON files and motion source files.
- `assets/profile/`
  Personal profile photos and social/meta images.
  Put your About image at `assets/profile/about-avatar.jpg` to replace the About Me portrait.
- `assets/tech-stack/`
  Icons used by the Portfolio tech stack tab.

Root-level public files should stay minimal:
- `robots.txt`
- `sitemap.xml`
- Google verification HTML files
- feature folders like `project-images/`
