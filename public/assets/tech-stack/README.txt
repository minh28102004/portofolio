Tech stack icons for the Portfolio tab live here.

You only need to drop an icon with the same base name and one of these extensions:
- `.svg`
- `.png`
- `.webp`
- `.jpg`
- `.jpeg`

The app will try files in that order automatically.

Recommended file names:

Frontend
- `html`
- `css`
- `javascript`
- `typescript`
- `react`
- `nextjs`
- `tailwind-css`
- `bootstrap`
- `material-ui`
- `ant-design`
- `shadcn-ui`
- `redux`
- `react-native`
- `flutter`

Backend
- `nodejs`
- `express`
- `nestjs`

Database & Services
- `mongodb`
- `postgresql`
- `sql-server`
- `firebase`

Tools & Platforms
- `git`
- `github`
- `axios`
- `postman`
- `insomnia`
- `figma`
- `vite`
- `vercel`
- `render`

Example:
- `assets/tech-stack/typescript.svg`
- `assets/tech-stack/postgresql.png`

If a file is missing, the UI will show a styled text fallback until you add the real icon.
